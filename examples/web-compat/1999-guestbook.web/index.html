Welcome, Web Traveler!

This original nostalgia fixture is here to exercise a classic guestbook flow: a simple HTML form posts to something that looks like old CGI, while Web.app safely serves a static shadow page.

No scripts run on the server. No network calls leave the bundle. The magic is only that a service worker can turn a POST into a static shadow page URL with query-string form fields.
