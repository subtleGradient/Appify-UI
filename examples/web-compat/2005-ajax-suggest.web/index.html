Glossary

AJAX

JavaScript asks the server for a small file and updates the page without reloading.

RSS

Feeds that made the whole web feel subscribable.

Weblog

A homepage that kept changing because people kept posting.

Ruby on Rails

A framework whose demos made database-backed apps feel oddly casual.

Tagging

Names added by users, not handed down by a taxonomy committee.

Podcast

Audio delivered through feeds and carried around in your pocket.

Mashup

A page that stitched together data from different places and made it feel new.

XML

Verbose, serious, and everywhere for a while.
