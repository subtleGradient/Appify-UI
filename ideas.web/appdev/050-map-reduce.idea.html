Converged Signals

Signal Reduce Consequence
Vibe Hackable native toybox, app garage, creative instrument. Do not make a dead settings editor. The product has to make working apps appear quickly and visibly.
Boundary .appdev is the private workshop; .app is shippable, inspectable, and reproducible. The source-of-truth rule must be explicit from day one.
First proof Self-hosting is emotionally important but should be a proof, not the entire architecture. Make Appify UI open its own .appdev, inspect its own target app, and run a tiny child app fixture.
Safety Signed runtime mutation is the sharp edge. Prefer edit-regenerate-run over mutate-signed-app-in-place.
Trust A document package can contain executable code. Opening a foreign .appdev should inspect only; run/build/sign/package actions should be explicit.
Delight The magic is inspectable causality: choice, generated file, visible preview, hackable result. Every magical action should reveal the file it touched.
