Context

The current professional web stack made complex production systems possible, but it also trained ordinary interactive creativity to feel like a specialist job. This dream wants the pendulum to swing back toward small, direct, joyful web artifacts.

start / values / interactive creativity / agent collaboration

Origin Story

This project comes from someone with deep history across React, React Native, MooTools, jQuery, ExtJS, Effect, and the long arc of web UI frameworks. That history matters because the rejection is not anti-web or anti-programming. It is a rejection of unnecessary ceremony becoming the default grammar for every tiny creative act.

Who This Is For

  • A writer who wants a story world, notes, maps, forms, and a public site without learning deployment culture.
  • An artist who wants a portfolio, playful sketch, or generative composition that can be copied like a document.
  • A parent who wants their children's creative output to remain portable and inspectable for decades.
  • An AI agent that needs a low-token, high-context workspace made of web-native files.
  • A future mobile Web.app that opens the same .web bundles on iOS and Android.

Enemy Model

The enemy is not any one framework. The enemy is the default path where a silly game, a living form, or a small interactive essay immediately becomes a Node project, a dependency graph, a deployment problem, and a set of insider norms.
