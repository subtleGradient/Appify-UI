Portability

The same bytes should feel natural as a local document, a static website, an emailed zip, a GitHub Pages site, a Vercel deployment, a removable-media archive, or a future mobile Web.app bundle.

Same Bytes Matrix

Place Should Work By Must Avoid
Web.app Double-clicking the package. Requiring a browser profile, dev server, or terminal ritual.
Static host Serving the same files with ordinary HTTP. Special server logic for core behavior.
Email or removable media Copying the package or zip. External cloud state as the only real document.
Future mobile Web.app Opening the same static bundle with a native document shell. Desktop-only build assumptions.

Offline Meaning

Offline support is not an afterthought. The artifact should still make sense when the internet is gone: a folder of pages, scripts, styles, assets, forms, and previews. Service Worker Terrain tracks the extra browser-managed state that appears when offline behavior moves from plain files into registered service-worker scopes.
