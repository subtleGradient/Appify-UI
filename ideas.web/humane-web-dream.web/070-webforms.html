WebForms

A .webform should feel like a PDF form made from the web: one file, visible controls, durable answers, and enough semantic structure for humans and agents to collaborate.

start / interactive creativity / agent collaboration / webform note

Contract

  • Saved answers live in HTML: value, checked, selected, textarea, and vetted editable HTML.
  • Runtime affordances can be injected by WebFormer, but the durable artifact remains readable HTML.
  • Clean diffs matter because git and agents are part of the working surface.
  • Custom elements are welcome when they mirror back to native controls or form-associated state that can be inspected.
