Jobs

Make

Start from a tiny working file, then change characters, rules, colors, and sounds.

Send

Zip it, email it, copy it, or host it as static files.

Inspect

Open the folder and read the source without project archaeology.

Mutate

Ask an agent to change one rule or add one sprite without rebuilding the universe.

Effort Pivot

The important threshold is not theoretical completeness. It is the moment when "make a little interactive thing" feels closer to making a document than starting a software project.
