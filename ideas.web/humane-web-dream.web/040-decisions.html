Decisions

These are the current working decisions. Future agents may revise them, but should not accidentally drift past them.

start / values / constraints / uncertainties

Working Decisions

  1. .web remains static, browser-native, and build-free.
  2. .webapp remains the place for packages that require npm, transforms, or dev servers.
  3. No index.html is required for a thought bundle; a directory listing can be the honest entrypoint.
  4. Interactive creativity starts with ordinary HTML, CSS, ESM, web components, relative assets, and browser APIs.
  5. Webforms store durable state in visible native HTML, not invisible sidecars.
  6. Reusable leverage should first appear as component-kit .web bundles, not a framework.
  7. Public, private, and internal surfaces should be separated by visible file and package shape before policy machinery.
  8. Agent collaboration should happen in the same web-native artifacts humans can inspect.

Not Yet Decisions

  • No decentralized identity protocol yet.
  • No registry yet.
  • No sync layer yet.
  • No universal app schema yet.
  • No claim that every arbitrary website should become a perfect local document.
