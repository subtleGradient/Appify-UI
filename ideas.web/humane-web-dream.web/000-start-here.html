Humane Web Dream

This bundle captures the dream: web-native creative artifacts that work as local documents, public websites, private workshops, agent collaboration surfaces, toys, games, forms, component kits, and long-lived family archives without turning every act of creativity into professional software engineering.

Map: values / context / current holds / decisions / constraints / interactive creativity / webforms / component kits / agent collaboration / portability / uncertainties / next probes / portable web trust idea

Intentional Directory Entry

This package intentionally has no index.html. The Web.app directory listing is the front door, because the shape of the files is part of the artifact. A future agent should see the little notes first, not a generated home page that hides the terrain.

Spine

plain web files
  -> single-file webforms
  -> interactive .web bundles
  -> reusable component-kit .web bundles
  -> local webspaces made of peer bundles
  -> public projections and private workshops from the same substrate
