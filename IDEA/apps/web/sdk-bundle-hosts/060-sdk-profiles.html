Per-SDK Profiles

The generic runner can be shared, but each SDK still needs a small profile for entry inference, canonical files, import aliases, save bridge, and default template.

Read order: index / problem / jobs / layers / package contract / dependencies / golf / probes

Profiles

Profile Default canonical file Entry inference Save risk Source anchor
cytoscape graph.cyjs or cytoscape.json Graph JSON first; otherwise index.html or src/main.*. Medium: graph state is clear, but app-specific panels and algorithms are not. Cytoscape.js docs
rete graph.rete.json nodes/registry.* plus graph JSON, then normal web entry. High: Rete explicitly leaves import/export to the app when objects are not plain JSON. Rete import/export
xyflow flow.xyflow.json Flow JSON with built-in nodes; custom nodes require src/nodes.*. Medium: toObject() gets nodes, edges, viewport, but not arbitrary component code. React Flow save/restore
tlcanvas document.tldraw.json or existing TLCanvas package files Snapshot files first; otherwise TLCanvas current package layout. Medium: snapshots have migrations, but assets and custom shapes need policy. tldraw persistence
d3 No universal graph file; source plus data are canonical. index.html, src/main.*, or appify.json entry. High: D3 is a visualization library, so the package code is the document behavior. D3 getting started

Bridge API Shape

window.appifyDocument = {
  kind: "xyflow",
  packageURL: "/doc/",
  readText(path),
  writeText(path, text),
  readAsset(path),
  writeAsset(path, blob),
  markDirty(reason),
  save(snapshot),
  updatePreview(blob)
}

Bridge Rules

  • Built-in templates use the bridge directly.
  • Normal packages can ignore the bridge and still render.
  • Packages that want native Save semantics implement or call the bridge adapter.
  • The bridge writes package-relative paths only.
  • Every SDK profile defines which files count as canonical, session, cache, and preview.

Profile-Specific Failure Messages

  • Cytoscaper.app: graph JSON loaded, but no style/layout found; using profile defaults.
  • Reter.app: graph contains non-JSON node type without a registered serializer.
  • React Flow Composer.app: custom node type declared in JSON but no renderer found.
  • TLCanvas.app: snapshot schema migrated; package preview needs regeneration.
  • Dee Three.app: no entry found; add index.html, src/main.js, or appify.json.
