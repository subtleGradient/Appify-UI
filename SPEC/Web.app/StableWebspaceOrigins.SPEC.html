Stable Webspace Origins

Web.app must separate the browser-visible origin from the private backend transport endpoint.

Invariant

WebKit sees the stable origin; the backend uses an ephemeral private port.

  • http://repo--hash.localhost:55555/... is the URL WebKit, page JavaScript, service workers, and native website storage see.
  • http://127.0.0.1:<ephemeral>/... is where the Runner/Bun backend may actually listen.
  • 55555 is origin identity, not the Runner listen-port requirement.
  • AppifyHost should map the visible origin to the hidden backend, preferably through WKWebsiteDataStore.proxyConfigurations on macOS 14 and newer.
  • Directly binding 55555 is only a temporary diagnostic fallback, not the stable design.
